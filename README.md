# PlaylistAppUpdate

This is an updated version of a class project. It's a very simple playlist app that allows users to search for songs, add them to their favorites, or add them to playlists they create. Uses Discogs and Youtube APIs for song searching. Heroku database for backend storage.

npm install to install needed packages, npm start to run the app

Fill out .env file following format of .env.example with necessary keys from database, Discogs, and Youtube.
